# Website - Meet and Mingle 

(English below)  

Este projeto é um site desenvolvido para o evento anual **Meet and Mingle** da empresa Kuraray South America, realizado em São Paulo com a participação dos escritório da América do Sul. Foi a primeira vez que o evento contou com uma página web oficial.

## Objetivo
Criar uma experiência digital simples e acessível para os participantes do evento, permitindo:
- Escolha de idioma (Português e Inglês).
- Visualização da programação completa.
- Acesso às informações dos palestrantes com link direto para seus perfis no LinkedIn.
- Interação por meio de um botão para envio de perguntas.

## Estrutura
### Página inicial:
  - Logo centralizada.
  - Mensagem de boas-vindas.
  - Seleção de idioma (PT/EN).
### Página principal:
  - Logo no topo.
  - Botão para enviar perguntas.
  - Programação detalhada por dia (horários, palestras e palestrantes).
  - Seção de palestrantes com:
    - Nome, foto e descrição.
    - Link para LinkedIn ao clicar na foto.

## Tecnologias Utilizadas
- *HTML5* para estrutura.
- *CSS3* para estilização.
- *GitHub Pages* para hospedagem.

## Como acessar
O site está disponível via GitHub Pages:  
[*Clique aqui para acessar*](https://anac-alcantara.github.io/Meet-Mingle/)

# Website - Meet and Mingle  

This project is a website developed for the annual **Meet and Mingle** event organized by Kuraray South America, held in São Paulo with participation from offices across South America. It was the first time the event featured an official web page.

## Purpose  
Create a simple and accessible digital experience for event participants, allowing:  
- Language selection (Portuguese and English).  
- Viewing the complete schedule.  
- Access to speaker information with direct links to their LinkedIn profiles.  
- Interaction through a button to submit questions.  

## Structure  
### Home Page:  
- Centered logo.  
- Welcome message.  
- Language selection (PT/EN).  

### Main Page:  
- Logo at the top.  
- Button to submit questions.  
- Detailed schedule by day (times, talks, and speakers).  
- Speaker section with:  
  - Name, photo, and description.  
  - LinkedIn link when clicking the photo.  

## Technologies Used  
- HTML5 for structure.  
- CSS3 for styling.  
- GitHub Pages for hosting.
